## Run Locally

Clone the project

```bash
  git clone git@github.com:Joachim-masson/WCS-Node2.git
```

Go to the project directory

```bash
  cd wcs-node2
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node index.js
```
